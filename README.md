# drasil
Static HTML website generator based on directory tree

This is a personal experiment, use it at your own risk :)

# Install

## Install from source

	cd drasil
	python setup.py install

## Create a tar ball

	cd drasil
	python setup.py sdist
	cd dist
	tar xzf drasil-*VERSION*.tar.gz
	cd drasil-*VERSION*
	python setup.py install
